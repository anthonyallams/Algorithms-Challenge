# CHALLENGE

    Sum all the prime numbers up to and including the provided number. A prime no is defined as a no greater than one and having only two diviors, 1 and itself. For example, 2 is a prime number because it's only divisible by one and two. The provided no may not be a prime.

## EXAMPLE

    sumAllPrimes(10) ==> 17
    sumAllPrimes(977) ==> 73156
